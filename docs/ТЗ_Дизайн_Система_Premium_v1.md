# ТЗ — Дизайн-система «Private & Premium» v1

Дата: 2026-07-07. Статус: **в работе**.
Связано с: [ТЗ_Дашборды_UX_v15](ТЗ_Дашборды_UX_v15.md). Источник цвета: `frontend/src/theme/ragPalette.ts`.
Реализация токенов: `frontend/src/theme/premium.ts`.

## 1. Цель и принципы
Приподнять восприятие интерфейса до уровня **private-banking** (дорого, спокойно, с воздухом),
**не теряя читабельность и юзабилити**. Светлая тема сохраняется, тёмная не вводится.

Принципы:
- **Спокойствие вместо крика.** Приглушённая RAG-палитра (шалфей/золото/терракот/графит) — как в v9.
- **Глубина через тени, а не через цвет.** Многослойные мягкие тени, тонкие хайрлайн-грани.
- **Воздух и ритм.** Больше отступов, единый радиус, выравнивание по сетке.
- **Золото — только акцент.** Тонкие линии/точки/логотип; никогда не заливка крупных зон.
- **Читабельность — приоритет.** Контраст текста к фону не ухудшается ради «красоты».

## 2. Токены (`theme/premium.ts`)
- **Радиусы:** карточки `16`, вложенные блоки `12`.
- **Тени:** `card`, `cardHover`, `raised`, `inset` — многослойные, мягкие.
- **Градиенты:** `sider` (глубокий графит-навигатор), `canvas` (тёплo-холодное полотно),
  `header`, `ink`, `goldLine`.
- **Золото (`GOLD`):** `base #B99A55`, `soft`, `line`, `glow`.
- **Грани:** `border #EAEBEE`, `borderStrong #DFE1E6`.
- **Поверхности:** `surfaceSoft`, `surfaceTint` (стеклянные подложки вложенных секций).

## 3. Компоненты и хелперы
- `premiumCard(accent)` → пропсы для AntD `<Card>`: радиус, хайрлайн-грань, мягкая тень,
  премиальная шапка; `accent` ∈ `ink|gold|sage|terracotta|slate|none` задаёт цвет акцент-маркера.
- `accentDot(color)` → акцентная риска перед заголовком карточки.
- `pageContainer`, `pageTitle`, `softPanel` → полотно/заголовок/подложка.
- `CollapsibleCard` → премиальная карточка со сворачиванием «в самом верху» (шеврон + заголовок),
  управляемый и неуправляемый режимы.

## 4. Применение
- **Сайдбар:** глубокий градиент-навигатор, золотой акцент логотипа, приглушённые заголовки групп.
- **Шапка:** белый градиент, тонкая нижняя грань, мягкая тень; заголовок с премиальным трекингом.
- **Карточки дашбордов:** `premiumCard`/`CollapsibleCard`, акцент-маркер по смыслу
  (пирог/динамика — `gold`/`slate`, риски — `terracotta`, «хорошо» — `sage`).
- **Диаграммы (ECharts):** мягкие линии, свечение точек-аномалий и пузырьков (`shadowBlur`),
  спокойные подписи, тонкая сетка.

## 5. Ограничения (анти-паттерны)
- Не вводить насыщенные/неоновые заливки крупных зон.
- Не жертвовать контрастом текста ради градиентов.
- Золото — не более чем на хайрлайнах/точках/иконках.
- Не ломать поведение AntD (только стилевые токены поверх).
